# Recording Instructions

Recommended duration: 3-5 minutes.

## Screen sequence

1. Show repository structure.
2. Show data acquisition command.
3. Show EDA notebook/figures.
4. Show model training and comparison output.
5. Show MLflow UI.
6. Show FastAPI Swagger UI.
7. Execute `/predict`.
8. Show Docker build/run evidence.
9. Show GitHub Actions successful run.
10. Show Render/Railway or Kubernetes deployment evidence.
11. Show `/metrics` endpoint and logs.

## Important

Do not show fake screenshots. If a deployment step is not executed, explain it as a prepared reproducible deployment route.
